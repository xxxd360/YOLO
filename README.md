## YOLO的使用
YOLO的初步体验流程上基本一致：
1：加载模型       2：用官方资源试检测预训练模型  
3：视频流的读取   4：用计数器获得图像   
5：使用Labelme做标签的准备与数据集配置文件的准备    6：模型的训练与试用保存权重
## YOLO原理的学习
学习了这种FPN的结构的网络模型  
yolov8除普通卷积层外使用了C2f模块与SPPF用于特征的提取与加强，SPPF用多尺寸池化核获得多尺度感受野。  
11n则将C2f替换回原来C3的优化版C3K2:C2f与C3K的核心都是运用了bottleneck，减少通道数的同时增加深度，提升特征的抽象性与减少计算：C2f更加轻量化，C3k2对尺寸更灵活精度更高但计算度稍高可通过控制bottleneck数量调节。  
12中使用了注意力机制：模块A2C2f将区域注意力与C2分残差结构结合
